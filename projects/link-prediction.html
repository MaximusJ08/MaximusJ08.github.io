Graph machine learning

Link prediction and ethical implications

A supervised graph ML project that predicts potential social network links while evaluating how model performance interacts with fairness across detected communities.

GitHub Open report Download report Back to projects

What I Built

The report describes a link prediction pipeline using structural, local, and community-aware graph features. Positive examples came from existing edges, while negative examples were sampled from node pairs without links. A Gradient Boosting classifier was identified as the strongest model, with community similarity emerging as an influential feature.

The second half of the project treats fairness as a first-class modelling concern. The analysis shows how graph structure can reinforce existing community advantages, then applies Equalized Odds post-processing to reduce unequal treatment across communities while acknowledging the resulting accuracy tradeoff.

Project Signals

  • Supervised link prediction over a social network graph.
  • Structural, local-neighbourhood, and community-aware feature engineering.
  • Gradient Boosting selected as the best-performing classifier.
  • Fairness evaluation using community-level TPR/FPR and Equalized Odds.
