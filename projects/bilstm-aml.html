BiLSTM multiple-choice NLP classifier

An applied NLP project for the AML "Read Between the Lines" Kaggle task, building a custom PyTorch BiLSTM answer selector and comparing it with a fine-tuned DistilBERT multiple-choice baseline.

What I Built

I implemented a multiple-choice text classifier that scores four candidate answers for each context-question pair. The custom pipeline builds a vocabulary from the training data, tokenizes context/question/answer combinations, pads sequences, tracks true sequence lengths, and trains a bidirectional LSTM with embeddings, dropout, packed padded sequences, cross-entropy loss, and best-model checkpointing.

Alongside the custom model, I adapted Hugging Face's multiple-choice framework to fine-tune DistilBERT as a transformer baseline. The notebooks cover train/validation splitting, metric tracking, hyperparameter tuning, inference over Kaggle test data, and submission-file generation for both the baseline and BiLSTM approaches.

Project Signals

  • Custom PyTorch BiLSTM multiple-choice answer scorer.
  • DistilBERT baseline using Hugging Face Trainer and tokenizer tooling.
  • Vocabulary building, sequence padding, length masking, and checkpointed training.
  • Kaggle inference workflow that exports prediction CSVs.
