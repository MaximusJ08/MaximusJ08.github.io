What I Built

The poster presents a sequence-modelling approach for a text classification problem, using a BiLSTM model as the central architecture. It connects the practical competition workflow with modelling decisions such as token handling, padding, bidirectional sequence learning, and Adam-based optimisation.

The project sits in the applied ML space: not just training a model, but communicating the model design, baseline context, and results in a concise visual format. The poster also links the implementation repository and the Kaggle task used for the experiment.

Project Signals

  • BiLSTM-based NLP classifier for a Kaggle text task.
  • Poster-format communication of model architecture and evaluation.
  • References transformer and DistilBERT context for comparison.
  • Reproducible project link included directly in the poster.
